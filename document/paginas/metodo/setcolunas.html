setColunas(['Vetor', 'de', 'elementos'])

Método que informa o nome das colunas do grid, lembrando que essa quantidade de colunas tem que ser simetrico com todas as funções e parâmetros do grid.

Parametros

  • Vetor de Elementos
    • Vetor com nome de todas as colunas do grid

Exemplo

var grid = new Grid('elementoID');
/* Lembrando que todos os parâmetros adiante tem que conter 3 colunas no caso do exemplo. */
grid.setColunas(['Nome', 'sobrenome', 'telefone']);
