JSON

Objeto principal para montagem do grid. O formato dele tem que corresponder exatamente como descrito na documentação

Parametros

  • Objeto
    • rows: Vetor que se refere as linhas.
    • id => id da linha
    • data => informações das colunas.

Exemplo

{
	rows: [
		{
			id: '1',
			data: ['1', '1', '1']
		},
		{
			id: '2',
			data: ['2', '2', '2']
		}
	]
}
