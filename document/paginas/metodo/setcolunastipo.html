setColunasTipo(['Vetor', 'de', 'tipo'])

Método com tipo de cada coluna para a ordenação do grid.

Parametros

  • Vetor de Tipo
    • Tipos: str => String / flt => Float / int => Inteiro.

Exemplo

var grid = new Grid('elementoID');
grid.setColunas(['Nome', 'sobrenome', 'telefone']);
grid.setLarguraColunas(['100', '*', '100']);
// Tipos das colunas
grid.setLarguraColunas(['str', 'str', 'int']);
