setColunasEventos(['Vetor', 'de', 'eventos'])

Método com evento de cada coluna.

Parametros

  • Tipos de Eventos
    • noEvent => Não possui eventos
    • check => Cria um checkbox. Callback: onCheck
    • edit => Cria um textfield na coluna para alteração. Callback: onEdit
    • button => cria botão. Parametros de configuração botão: setColunasBotoes

Exemplo

var grid = new Grid('elementoID');
grid.setColunas(['Nome', 'sobrenome', 'telefone']);
grid.setLarguraColunas(['100', '*', '100']);
grid.setLarguraColunas(['str', 'str', 'int']);
// Eventos da coluna
grid.setColunasEventos(['noEvent', 'edit', 'noEvent']);
