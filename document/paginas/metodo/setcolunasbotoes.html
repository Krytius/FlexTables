setColunasBotoes(['Vetor', 'de', 'parametros', 'dos', 'botoes'])

Método com parametro de configuração dos botões do grid. Lembrando que a posição do vetor tem que corresponder a posição dos eventos da colunas.

Parametros

  • Tipos de Parametros
    • classe => classe CSS. (Não obrigatório)
    • id => ID. (Não obrigatório)
    • html => Interior do botão.
    • evento => Callback do click do botão. callbackDoBotao. (Não obrigatório)

Exemplo

var grid = new Grid('elementoID');
grid.setColunas(['Produto', 'Preço', 'Quantidade']);
grid.setLarguraColunas(['100', '*', '100']);
grid.setLarguraColunas(['str', 'str', 'int']);
grid.setColunasEventos(['noEvent', 'button', 'noEvent']);
grid.setColunasBotoes([{},{
  classe: 'button2',
  id: 'button2',
  evento: function(id, elemento) {},
  html: "Teste"
},{}]);
