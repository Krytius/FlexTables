onAdicionar

Evento do monitor de eventos que pega qualquer adição nas linhas do grid.

Callback

  • Id - Id da nova linha adicionada.
  • Data - Vetor das colunas do grid, cada posição representa uma coluna.

Exemplo

grid.monitorDeEventos("onAdicionar", function(id, data) {
	console.log(id, data);
});
